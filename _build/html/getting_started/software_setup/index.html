Software Setup¶

This section has three sections:

  • Host/Laptop Setup
  • Jetson Setup
  • Bringing the Host and Jetson Together
Host setup details how to install ROS and the simulators on your laptop.
Jetson setup goes over how to set up the Jetson TX2.
Combine setup goes over how to communicate between your host laptop and the TX2 once you have already completed the above two sections.
